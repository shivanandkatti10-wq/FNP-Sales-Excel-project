## FNP - Sales Analysis Project

## Project Overview
This repository presents a complete data analysis project built in Microsoft Excel, demonstrating an end-to-end workflow from raw dataset to actionable insights and interactive dashboard. The focus of the project is to illustrate best practices for real-world data analysis using Excel’s built-in features along with AI-assisted tools to enhance efficiency and insight generation.

## Key Features

## ✔️ Full Data Lifecycle

* Data import and organization

* Data cleaning and preprocessing (handling missing values, formatting)

* Exploratory Data Analysis (EDA) with descriptive statistics

* Interactive dashboard featuring PivotTables, slicers, and charts

* Visual storytelling for trends and key performance indicators (KPIs)

# ✔️ Skills Applied

* Excel Power Query

* PivotTables & PivotCharts

* Dynamic formulas (SUMIFS, COUNTIFS, etc.)

* Conditional formatting for insights

* Interactive dashboard design
